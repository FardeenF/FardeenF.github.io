Mining Matters -
Core Shack Escape Room (Unity)

The Core Shack Escape Room is a 3D point and click virtual escape room made in Unity for WebGL.
It was developed as a capstone project for Mining Matters, a non-profit charitable organization.
This is the third game in a trilogy of escape games offered by Mining Matters.

Ontario Tech VS
The World (Unity)

Ontario Tech VS. The World is a rhythm-based, comic styled fighting game
that utilizes volumetric models recorded by staff and students. This project was made with
the volumetric technology developed by the Soar team and made in Unity.

SkyBound (C++)

SkyBound is a 3D adventure/puzzle game made using OpenGL.
The game was created in a custom-made engine in C++.

Fury of the Dead (C++)

Fury of the Dead is a 2D sprite based wave-survival game where
the player is tasked with defeating zombies and getting the highest
possible wave count. This game was created in C++.

One Shot, One Chance (Unity)

One Shot, One Chance was a game developed for the GDSA Fall 2021 Game Jam
in 72 hours made in Unity. It is a 2D puzzle game that has the player try and shoot
projectiles to collect gems to clear levels with only one shot.
